# Pilot’s Guide to Medical Certification

A web-based educational platform designed to demystify the FAA medical certification process for new pilots.

## 📖 Description

### What is this project?
This project creates an educational web platform that guides new pilots through the FAA medical certification process. It provides clear, accurate, and accessible information, including interactive tools to reduce confusion and stress when seeking a medical certificate.

### Who is it for?
- **Primary Client:** The FAA’s Civil Aerospace Medical Institute (CAMI) department.
- **End Users:** New pilots who may not understand the FAA medical certificate requirements, necessary disclosures, and how to navigate the medical exam process with confidence.

## 🚀 Getting Started

### Prerequisites
- **Node.js** (v18 or higher)
- **npm** (typically bundled with Node.js)

### Installation & Running

1.  **Clone the repository**
    ```bash
    git clone <your-repository-url>
    cd pilots-medical-guide
    ```

2.  **Navigate to the frontend directory and install dependencies**
    ```bash
    cd frontend
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```

4.  **Open your browser**
    The application will typically be available at `http://localhost:8443`. It runs entirely in the browser using mock data for development.

## ✨ Key Features

-   **Informative Subsections:** Drop-down menus explaining medical abbreviations and their roles in the certification process.
-   **Responsive Design:** Dynamic formatting that works seamlessly on both mobile and desktop browsers.
-   **Interactive Checklists:** Guided checklists for specific medical conditions to help pilots prepare.
-   **Reduced Cognitive Load:** Information is segmented and abstracted to prevent user overwhelm.
-   **Future Idea:** Interactive human body diagram to show conditions that impact specific body parts relevant to an Aviation Medical Examiner (AME).
